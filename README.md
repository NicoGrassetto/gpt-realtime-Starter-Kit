# GPT-4o Realtime Starter Kit

A starter kit for building realtime applications with GPT-4o.

## Project Structure

```
├── assets/          # Static assets
├── frontend/        # Frontend application
├── hooks/           # Lifecycle hooks
├── infra/           # Infrastructure as code
├── src/             # Backend source code
│   └── main.py      # Application entry point
├── LICENSE
├── README.md
└── requirements.txt
```

## Getting Started

### Prerequisites

- Python 3.10+

### Installation

```bash
pip install -r requirements.txt
```

### Running

```bash
python src/main.py
```
